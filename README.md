# Goducate Sabah Website

Simple static educational website with 6 pages and JSON-based lesson cards.

## Update lesson links (admin)

1. Open `assets/data/lessons-data.js`.
2. Find the section you want to edit:
   - `beginner`
   - `primary1`
   - `primary2`
   - `primary3`
   - `advanced`
3. For each lesson item, update:
   - `title`
   - `description`
   - `link` (Google Drive URL)
4. Save and refresh the corresponding page.

No HTML edits are needed for normal lesson updates.
